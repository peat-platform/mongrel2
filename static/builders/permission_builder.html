This page is designed so that you can easily create an openi permissions file to include in your mobile application. To create follow these steps:
  1. Either paste a permissions object or text containing OPENi type ids in to the input box on the left and click on start editing.
  2. The system will automatically extract the type ids and create editable intefaces in the center column
  3. Select checkboxes to set access level and type; the final permissions file is updated as you go
  4. Click on the 'Download Permissions File' when you are happy with your choices.
