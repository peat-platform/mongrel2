What is PEAT

This is PEAT, PEAT stands for Personal Data, Apis, and Trust. PEAT allow your users manage their personal information with the PEAT cloudlet framework across many mobile applications. PEAT's advanced REST API framework connects your users to a multitude of cloud-based services. PEAT allows your users control their data safeguarded by a standards based compliant security framework.

Personal Data

PEAT aims at inspiring innovation in the European mobile applications industry, by radically improving the interoperability of cloud-based services and trust in personal cloud storage through the development of a consumer-centric, open source mobile cloud applications platform.

Client Libraries

To experiment with PEAT's Graph API, security, and cloudlet frameworks you can use the REST Endpoints HTML UI.

  • Graph API REST Endpoints HTML UI
  • Cloudlet REST Endpoints HTML UI
  • Security API REST Endpoints HTML UI

To allow Android application development we provide a mobile client library. This library abstracts and simplify access to the PEAT services, in simple terms it aggregates the REST clients of the three frameworks, but it also adds a significant number of enhancements including a number of security mechanisms and UI components.
